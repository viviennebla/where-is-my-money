## 统计维度

| 维度 | 数据来源 | 说明 |
|------|----------|------|
| **月度汇总** | SUM by type + date | 月支出/收入/结余 |
| **类别构成** | `source_category` | WeChat 的"商户消费""转账""红包"等，v0.4 后切换标签 |
| **商户排名** | `merchant_name` | 按支出金额 Top N |
| **账户概览** | accounts 表 | 当前余额 |
| **趋势图** | 按月聚合 | 近 6/12 月走势 |

## 后端 API

| 端点 | 参数 | 返回 |
|------|------|------|
| `GET /stats/summary` | `year`, `month` | `{expense, income, net, tx_count}` |
| `GET /stats/monthly-trend` | `months` (default 6) | `[{year, month, expense, income}]` |
| `GET /stats/category-breakdown` | `year`, `month`, `type` | `[{category, amount, count, pct}]` |
| `GET /stats/merchant-ranking` | `year`, `month`, `limit` | `[{merchant, amount, count}]` |

## 技术选型

- **图表库：Recharts** — React 原生，轻量，声明式 API
- **新增后端：** `routers/stats.py`
- **新增前端：** `components/dashboard/StatCards.tsx`, `MonthlyTrend.tsx`, `CategoryPie.tsx`, `MerchantRanking.tsx`, `AccountBalances.tsx`
- **改造：** `pages/DashboardPage.tsx` 从简单流水列表改为看板首页
- **首页布局：** 看板在上面，交易列表保留在下方（可折叠）

## 其他可选方案

| 方案 | 说明 | 优先级 |
|------|------|--------|
| 日历热力图 | 每日支出金额→颜色深浅，类似 GitHub 贡献图 | P2 |
| 周/月环比 | 本周 vs 上周，本月 vs 上月 | P2 |
| 大额标记 | 单笔 > 均值 3 倍自动标出 | P3 |
| 消费时段分布 | 早/中/晚/深夜 金额分布 | P3 |
| 退款率 | 退款金额 / 总支出 | P3 |
